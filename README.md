Personal portfolio website for Pedro Henrique Maciel — backend engineer focused on Java, Spring Boot and scalable APIs.

🔗 **Live:** [pedromaciel.dev](https://pedromaciel.dev)

Built with vanilla HTML, CSS and JavaScript — no frameworks, no build step, single `index.html`.
