// 01 — Sobre

Quem Sou

Sou Pedro Henrique Maciel, desenvolvedor backend baseado no Rio de Janeiro, Brasil, focado em construir sistemas bem estruturados e de alta qualidade.

Minha stack principal gira em torno de Java e Spring Boot, com experiência sólida em APIs REST, bancos relacionais com PostgreSQL e arquiteturas orientadas a microsserviços.

Me importo profundamente com código limpo, princípios SOLID e escrever software fácil de ler, testar e manter — não apenas software que funciona.

Atualmente expandindo para DevOps, Docker e infraestrutura cloud para me tornar um engenheiro backend mais completo.
